How I work

Three principles, no exceptions.

Every engagement runs through the same logic. Each principle exists because skipping it has cost someone, somewhere, money they did not need to spend.

01

Audit first

The €300 systems audit is the only way in. Nothing is built before that. The audit defines what to build and in what order, and the fee is deducted from the project if it proceeds.

02

Define done before we start

If we cannot write down what finished looks like, the project does not start. Scope without a definition of done is how engagements run forever and clients lose trust.

03

I own the system. You own the business.

Every system is documented, transferable, and maintainable by another developer. No lock-in. No hidden dependencies. You can take the build elsewhere the day it ships if that is what you need.
